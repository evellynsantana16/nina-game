# Nina - Meu Gatinho (Vercel)

Versão 100% estática em HTML, CSS e JavaScript.

## Publicar na Vercel

1. Extraia a pasta.
2. Entre em vercel.com.
3. Crie um projeto e envie esta pasta ou conecte um repositório GitHub.
4. Framework Preset: Other.
5. Build Command: deixe vazio.
6. Output Directory: deixe vazio.
7. Clique em Deploy.

## Recursos

- Responsivo para celular, tablet e computador.
- Menu inferior no celular.
- Nina preta e branca com olhos verdes.
- Fome, higiene, energia, diversão e amor.
- Moedas, XP, níveis e recompensa diária.
- Cozinha, quarto, banheiro e loja.
- Minijogo Caça ao Ratinho.
- Salvamento por aparelho usando localStorage.
- PWA: pode ser adicionada à tela inicial do celular.
- Cache offline após a primeira visita.

Observação: como não existe login nem banco, cada celular possui seu próprio progresso.
